Gradio Ticket Systeem

Automatisch tickets genereren met de Google Sheets API

Dit project was gestart voor een evenement gedurende mijn stageperiode in 2024.

Genodigde vulden een korte vragenlijst in, deze data kwam terecht in een database. Vervolgens werd, met behulp van de Python Library 'Pillow', een unieke QR code gegenereerd. Door middel van de Gmail API werd vervolgens een unieke mail gestuurd, met de QR code op een digitale ticket. 
Verder werd automatisch bijgehouden naar wie al tickets waren gestuurd, zodat er geen dubbele mails zouden ontstaan. 

Bij binnenkomst van het evenement werden alle tickets gescand. Op deze manier kon aanwezigheid gemakkelijk worden bijgehouden.
